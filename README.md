# Nova-Dust Discord RPG Bot

Post-apocalyptic RPG Discord bot featuring:

- Player registration
- Survivor role assignment
- XP & leveling
- NovaDust currency
- Shop & inventory
- Adventure encounters
- Gambling

## Setup

1 Install Node.js

2 Install dependencies

npm install

3 Create .env

DISCORD_TOKEN=your_token
DATABASE_URL=render_postgres_url

4 Run locally

node index.js

## Deploy to Render

1 Push this repo to GitHub
2 Create Render Web Service
3 Connect GitHub repo
4 Add Environment Variables

DISCORD_TOKEN
DATABASE_URL

Deploy.

Your Nova-Dust bot will run 24/7.
